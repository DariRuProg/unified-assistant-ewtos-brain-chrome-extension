Vault verknupfen

Ein Vault ist dein Obsidian-Ordner oder ein beliebiger Ordner mit Markdown-Dateien. EwtosBrain nutzt ihn als Wissensbasis fur den Chat.

Absoluter Pfad zum Ordner. Der Server muss Lesezugriff auf diesen Pfad haben.

Kein Vault? Du kannst das auch spater in den Einstellungen nachholen. YouTube-Transcripts und Tools funktionieren auch ohne Vault.
