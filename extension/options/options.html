video-brain & License

Spiegelt fertig analysierte Videos in deine eigene Supabase, damit die video-brain Mobile-App sie lesen kann. Jeder Kunde bringt sein eigenes Supabase-Projekt mit — deine Videodaten bleiben bei dir.
Z.B. https://abcdefgh.supabase.co — zu finden unter Project Settings → API.
Öffentlicher Leseschlüssel — wird von der video-brain PWA genutzt. Project Settings → API → anon public.
Schreibschlüssel für den Sync (bypassed RLS). Wird serverseitig gespeichert, nie an die PWA weitergegeben. Project Settings → API → service_role secret.
Die UUID des Supabase-Auth-Users der video-brain PWA nutzt (Authentication → Users). ewtos-brain setzt user_id auf diesen Wert, damit die PWA die Daten per RLS sehen kann.
Von ewtos.com ausgestellter Lizenz-Key für video-brain. Wird vor jedem Sync gegen die zentrale Lizenzdatenbank geprüft.

Connected Vaults

Jeder Vault ist ein Ordner mit .md-Dateien (Karpathy-Style). Pro Vault eigener System-Prompt + eigener Chat-Verlauf. Im Side-Panel wechselst du zwischen den Vaults.

Blueprints

Vault-Vorlagen. Built-in liefert EwtosBrain mit; importierte Blueprints kommen von dir oder Premium-Quellen.
JSON-Datei mit Blueprint-Struktur. Unsignierte Blueprints werden vor dem Import angefragt.
