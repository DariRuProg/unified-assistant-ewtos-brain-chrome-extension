# EwtosBrain — Features

EwtosBrain ist eine Chrome-Extension + ein selbst betriebener Server, der deinen
Obsidian-Vault per KI nutzbar macht.

## Was es kann
- **Vault-Chat** nach der [[karpathy-methode]] — frag dein Wissen direkt im Browser.
- **Browser-Tools:** Seite als Markdown clippen, SEO-Check, Bild-Analyse, Screenshot
  mit Annotation, Farb-Pipette, Link-Extraktor, YouTube-Transkripte.
- **Note-Taker:** Todos, Scratchpad, Bookmarks.
- **Multi-LLM (BYOK):** Claude, OpenAI, Mistral oder Ollama (lokal) — dein Schlüssel.

## Daten & Kontrolle
- **Self-hosted:** der Server läuft bei dir (lokal oder auf deinem VPS). Inhalte
  verlassen deine Infrastruktur nicht in fremde Cloud.
- **DSGVO-freundlich:** mit Ollama bleibt sogar das LLM komplett lokal.

## Demo vs. echtes Produkt
Diese Demo ist **read-only** mit Beispiel-Daten. Für den eigenen Vault + Schreiben
installierst du den kostenlosen EwtosBrain-Server (Download auf ewtos.com).
